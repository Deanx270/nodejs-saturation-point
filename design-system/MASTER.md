# The Saturation Point - Design System (UI/UX Pro Max)

## Brand Identity
A premium, professional e-commerce store specializing in high-end fountain pens, inks, and papers. The visual language should reflect sophistication, craftsmanship, and elegance.

## Color Palette
*   **Primary Background:** `#FAF9F6` (Warm Alabaster / Ivory) - Used for page backgrounds to give a soft, premium feel.
*   **Secondary Accent:** `#1B263B` (Deep Blue Ink) - Used for primary buttons, footers, and strong contrast elements.
*   **Accent / CTA:** `#D4AF37` (Classic Gold) - Used sparingly for call-to-action buttons, active states, and highlights.
*   **Text (Primary):** `#2D3436` (Charcoal) - Used for primary headings and body text to soften contrast against the ivory background.
*   **Text (Secondary/Muted):** `#636E72` (Slate Gray) - Used for secondary text, metadata, and disabled states.

## Typography
*   **Headings:** `Cormorant Garamond` (Google Fonts) - A classic, elegant serif font representing tradition and fine writing.
*   **Body:** `Inter` (Google Fonts) - A clean, highly readable sans-serif font for product descriptions, UI elements, and data.

## Currency Format
*   Philippine Peso (₱)
*   Format: `₱#,##0.00`

## Components & Styling
### Buttons
*   **Primary Button:** Deep Blue Ink (`#1B263B`) background, White text, slight border radius (4px), smooth hover transition to a slightly lighter blue.
*   **Accent Button (Buy/Checkout):** Classic Gold (`#D4AF37`) background, Charcoal text, hover effect that slightly brightens the gold.
*   **Ghost Button:** Transparent background, Deep Blue Ink border and text.

### Cards (Products/Brands)
*   White (`#FFFFFF`) or slightly lighter Ivory background.
*   Subtle shadow: `box-shadow: 0 4px 6px rgba(0, 0, 0, 0.05);`
*   Hover effect: Slight lift (`transform: translateY(-2px);`) and increased shadow.
*   Image area: Minimalist framing, generous padding.

### Forms
*   Clean, minimalist inputs.
*   Bottom border only or full very light border.
*   Focus state: Outline color changes to Deep Blue Ink (`#1B263B`).
*   Labels in small uppercase `Inter` for an editorial look.

### Spacing & Layout
*   Embrace whitespace.
*   Use a generous grid system.
*   Padding inside components should feel luxurious and uncrowded.
